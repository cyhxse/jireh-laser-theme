# Jireh Laser WordPress Theme

专业的激光设备制造商WordPress主题

## 主要特性

- 响应式设计
- SEO优化
- 产品展示功能
- 新闻管理
- 多语言支持
- 自定义主题选项

## 安装说明

1. 下载主题zip包
2. 登录WordPress后台
3. 进入 外观 > 主题 > 添加
4. 上传主题zip包并启用

## 演示数据

主题包含完整的演示数据，包括：
- 产品展示
- 公司介绍
